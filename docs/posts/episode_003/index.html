Summary

自然言語をよしなに可視化・分析できるライブラリ「nlplot」を公開したよ、という話をしました。

nlplotについて

pipでインストールできます。

pip install nlplot

下記グラフが簡単に描画できるので、データの概要・傾向を掴みたいときに便利かと思います。

  1. N-gram bar chart
  2. N-gram tree Map
  3. Histogram of the word count
  4. wordcloud
  5. co-occurrence networks
  6. sunburst chart
  7. pyLDAvis
